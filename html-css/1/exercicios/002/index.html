Paragráfos e quebras de linha

Você pode escrever um parafráfo de de qualquer jeito, desde que esteja dentro da tag paragráfo, isto é muito útil para deixar o código mais organizado.

Se caso for necessário fazer quebra de linha é só usar a tag <br>.
